Day 19 – Flask Framework Basics

Overview

This module introduces the Flask web framework and demonstrates how to build a basic backend server with API routes and JSON responses.

Learning Objectives

- Understand Flask framework basics

- Run a local development server

- Create route endpoints

- Handle HTTP requests

- Return JSON responses

Technologies Used

- Python

- Flask

Implemented Features

    - Home route (/)

    - JSON API endpoint (/employees)

    - Local server execution

    - Debug mode enabled

How to Run

    Install Flask:

        pip install flask


    Run the application:

        python flask_intro.py


    Open browser:

        http://127.0.0.1:5000


Example JSON Response

{
  "status": "success",
  "data": [
    {
      "emp_id": 1,
      "name": "John",
      "department": "IT"
    }
  ]
}
