## Day 27 – Sorting & Ordering Data

# Overview

This module demonstrates sorting functionality in APIs and shows how to combine sorting with filtering and search operations.

# Learning Objectives

Understand sorting in APIs
Use query parameters for sorting
Implement ascending & descending order
Combine sorting with filters & search
Build flexible query systems

# Key Concepts

Sorting:

Arranging data in a specific order.

SQL ORDER BY:

SELECT * FROM employees ORDER BY name ASC;
SELECT * FROM employees ORDER BY name DESC;

# ASC vs DESC

ASC → Ascending order
DESC → Descending order

# Features Implemented

Sorting by field (name)
Ascending & descending order
Search functionality
Department filtering
