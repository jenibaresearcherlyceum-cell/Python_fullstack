## Day 25 – Pagination & Efficient Data Handling

# Overview

This module demonstrates pagination and filtering techniques to efficiently handle large datasets in backend systems.

# Learning Objectives
Understand pagination concepts
Limit API response size
Use query parameters
Filter data dynamically
Improve API performance

# Key Concepts
Pagination

Splitting large data into smaller chunks.

Query Parameters

Used to control API response:

?page=1&limit=5
Filtering

Returning only specific data based on conditions.

# Features Implemented
Pagination logic
Query parameter simulation
Department-based filtering

# Example Response
{
  "status": "success",
  "page": 1,
  "limit": 2,
  "data": [
    {
      "id": 1,
      "name": "John",
      "department": "IT"
    }
  ]
